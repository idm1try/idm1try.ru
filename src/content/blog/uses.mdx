## CLIs

- [**neovim**](https://neovim.io) - My main code editor, with lua configuration
- [**nix**](https://nixos.org) - Best package manager. I use with flakes,
  home-manager and nix-darwin
- [**btop**](https://github.com/aristocratos/btop) - System monitoring
- [**starship**](https://starship.rs) - Customizable cross-shell prompt
- [**zoxide**](https://github.com/ajeetdsouza/zoxide) - Smarter `cd` command
- [**bat**](https://github.com/sharkdp/bat) - A better `cat` with syntax
  highlight
- [**exa**](https://the.exa.website) - A better `ls`
- [**dust**](https://github.com/bootandy/dust) - A better `du`
- [**ripgrep**](https://github.com/BurntSushi/ripgrep) - A better `grep`
- [**fzf**](https://github.com/junegunn/fzf) - Fuzzy finder
- [**zsh**](https://zsh.org) - My main shell. With these plugins:
  - [**Oh my zsh**](https://ohmyz.sh) - Plugins: `sudo`, `git`, `extract`
  - [**zsh-nix-shell**](https://github.com/chisui/zsh-nix-shell)
  - [**zsh-autosuggestions**](https://github.com/zsh-users/zsh-autosuggestions)
  - [**fast-syntax-highlighting**](https://github.com/zdharma-continuum/fast-syntax-highlighting)
  - [**zsh-autopair**](https://github.com/hlissner/zsh-autopair)
